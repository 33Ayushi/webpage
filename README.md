Magic Envelope 💌

A simple and interactive web project where a digital envelope opens to reveal a special message.
This project creates a small emotional experience by simulating the feeling of opening a real envelope and discovering a hidden message inside.

✨ Features

Interactive envelope opening animation

Hidden message reveal

Simple and clean UI

Built using HTML, CSS and JavaScript
🚀 How to Run

Clone the repository

git clone https://github.com/yourusername/magic-envelope.git

Open the project folder

Run index.html in your browser
